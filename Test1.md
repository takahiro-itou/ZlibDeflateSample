```

```

##  ヘッダ

- 78 9C

```
・0111 1000  1001 1100
↓
・0111 1000 10 0 11100

0111  : Window Size = 7 => 2^(7+8) = 32k
1000  : 圧縮モード 8 (Z_DEFLATE)
10    : 圧縮レベル 2
0     : プリセット辞書 (なし)
11100 : ヘッダチェック
```
